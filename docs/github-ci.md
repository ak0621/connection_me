# GitHub CI Packaging

This repository includes a GitHub Actions workflow at `.github/workflows/build.yml`.
The workflow builds the same C++20/CMake codebase on Linux, macOS, and Windows runners and uploads platform-specific packages.

## Triggering the Workflow

The workflow runs automatically on:

- every push to GitHub
- every pull request
- manual runs from the GitHub Actions tab

To trigger it manually:

1. Open the GitHub repository in a browser.
2. Go to **Actions**.
3. Select the **build** workflow.
4. Click **Run workflow**.
5. Choose the branch, usually `main`, and start the run.

## Downloading Installers

After the workflow finishes:

1. Open the completed workflow run.
2. Scroll to **Artifacts**.
3. Download the artifact for the target platform.

Expected artifacts:

| Artifact | Runner | Installer inside artifact |
|----------|--------|---------------------------|
| `mybarrier-linux-x86_64` | `ubuntu-24.04` | `mybarrier-*-Linux-*.deb` |
| `mybarrier-macos-arm64` | `macos-15` | `mybarrier-*-Darwin-*.dmg` |
| `mybarrier-windows-x86_64` | `windows-2022` | `mybarrier-*-Windows-*.exe` |

Each artifact also contains a runnable binary directory under `dist/` and archive packages such as `.zip` or `.tar.gz` when generated by CPack.

If a platform job fails, the workflow uploads a `*-diagnostics` artifact with `configure.log`, `build.log`, or `package.log` from the failing runner.

## Why CI Is Needed

The Ubuntu development host can build the Linux executable and `.deb` package only. macOS installers require the Apple toolchain and DMG tooling from a macOS runner. Windows installers require a Windows executable plus NSIS, which the workflow installs on the Windows runner before running CPack.

## Local Packaging Commands

Linux/macOS:

```bash
scripts/package-current.sh
```

Windows PowerShell:

```powershell
.\scripts\package-current.ps1
```

Direct CPack command:

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release -j
cmake --build build --target package --config Release
```
